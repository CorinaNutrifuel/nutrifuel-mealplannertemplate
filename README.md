# NutriFuel AI Meal Planner

An AI-powered meal planning configurator built as a team MVP for an AI / Machine Learning project.

## About the Project

NutriFuel AI Meal Planner is a React-based web application designed to generate personalized 7-day meal plans based on a client’s profile, nutritional strategy, food preferences, and practical cooking needs.

The system allows the user to input client data such as age, height, weight, sex, objective, preferred foods, foods to avoid, cooking level, meal structure, and meal-prep preferences. It also calculates BMR using the Mifflin-St Jeor formula and uses calorie, protein, carbohydrate, fat, and fiber targets to guide the meal plan generation.

## Core Features

- Client profile configuration
- BMR calculation
- Nutrition strategy setup
- Calorie and macronutrient targets
- Fiber target validation
- Meal structure customization
- Meal-prep planning
- Anti-food-waste recipe logic
- Recipe library with breakfast, snack, lunch, and dinner options
- AI-generated 7-day meal plan
- Automatic validation against nutrition targets
- Saved client profiles

## MVP Status

This project is currently at MVP stage. The core functionality is working: users can configure a client profile, define a nutrition strategy, and generate a structured 7-day meal plan using AI.

The system also includes validation logic to keep daily calories and macronutrients within an accepted range, making the output more aligned with the nutrition strategy.

We plan to continue improving the platform, especially in terms of accuracy, user experience, scalability, and future integration possibilities.
